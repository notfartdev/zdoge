# Recent Features & Updates

## User Experience

**Smart Error Messages**
Context-aware error suggestions across all operations.

**Transaction History**
Local transaction history with filtering by type and block explorer links.

**Note Management**
View, filter, and sync shielded notes with blockchain.

**Estimated Fees**
Transaction fee preview before confirmation.

**Progress Tracking**
Real-time progress indicators for proof generation and transaction status.

## Privacy

**Timestamp Rounding**
Timestamps rounded to 5-minute buckets to reduce timing correlation.

**Minimize Event Data**
Only essential data stored in backend indexer.

**Memo Size Caps**
128-byte cap on encrypted memos enforced in backend.

## Security

**Enhanced Input Validation**
Comprehensive parameter validation including amount bounds, proof format, and token addresses.

**Rate Limiting**
IP-based rate limiting on backend endpoints.

**Encrypted Local Storage**
XSS protection through encrypted localStorage.
